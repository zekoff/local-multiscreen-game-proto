# Asset credits

All image assets under `public/assets/` are **CC0 1.0 (Public Domain)** — no
attribution is required, but their provenance is recorded here for auditability.
These load **only on the main screen in Phaser renderer mode** (`?renderer=phaser`);
the phone consoles and the Canvas 2D renderer use none of them.
